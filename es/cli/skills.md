# skills

Inspecciona habilidades (empaquetadas + espacio de trabajo + anulaciones gestionadas) y consulta qué es elegible frente a los requisitos faltantes. Relacionado:

-   Sistema de habilidades: [Habilidades](../tools/skills.md)
-   Configuración de habilidades: [Configuración de habilidades](../tools/skills-config.md)
-   Instalaciones de ClawHub: [ClawHub](../tools/clawhub.md)

## Comandos

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
