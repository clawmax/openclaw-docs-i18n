# setup

Inicializa `~/.openclaw/openclaw.json` y el espacio de trabajo del agente. Relacionado:

-   Primeros pasos: [Primeros pasos](../start/getting-started.md)
-   Asistente: [Incorporación](../start/onboarding.md)

## Ejemplos

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Para ejecutar el asistente mediante setup:

```bash
openclaw setup --wizard
```
