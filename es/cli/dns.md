# dns

Ayudantes de DNS para descubrimiento de área amplia (Tailscale + CoreDNS). Actualmente enfocado en macOS + CoreDNS de Homebrew. Relacionado:

-   Descubrimiento de puerta de enlace: [Descubrimiento](../gateway/discovery.md)
-   Configuración de descubrimiento de área amplia: [Configuración](../gateway/configuration.md)

## Configuración

```bash
openclaw dns setup
openclaw dns setup --apply
```
