# Funciones

## Aspectos destacados

## Canales

WhatsApp, Telegram, Discord e iMessage con una única Puerta de Enlace.

## Plugins

Añade Mattermost y más con extensiones.

## Enrutamiento

Enrutamiento multiagente con sesiones aisladas.

## Medios

Imágenes, audio y documentos de entrada y salida.

## Aplicaciones e Interfaz de Usuario

Interfaz de Control Web y aplicación complementaria para macOS.

## Nodos móviles

Nodos para iOS y Android con emparejamiento, voz/chat y comandos avanzados para dispositivos.

## Lista completa

-   Integración de WhatsApp vía WhatsApp Web (Baileys)
-   Soporte para bots de Telegram (grammY)
-   Soporte para bots de Discord (channels.discord.js)
-   Soporte para bots de Mattermost (plugin)
-   Integración de iMessage vía CLI local imsg (macOS)
-   Puente de agente para Pi en modo RPC con transmisión de herramientas
-   Transmisión y fragmentación para respuestas largas
-   Enrutamiento multiagente para sesiones aisladas por espacio de trabajo o remitente
-   Autenticación por suscripción para Anthropic y OpenAI vía OAuth
-   Sesiones: los chats directos se agrupan en un `main` compartido; los grupos están aislados
-   Soporte para chats grupales con activación basada en menciones
-   Soporte multimedia para imágenes, audio y documentos
-   Gancho opcional para transcripción de notas de voz
-   WebChat y aplicación de barra de menús para macOS
-   Nodo iOS con emparejamiento, Canvas, cámara, grabación de pantalla, ubicación y funciones de voz
-   Nodo Android con emparejamiento, pestaña Connect, sesiones de chat, pestaña de voz, Canvas/cámara/pantalla, más comandos de dispositivo, notificaciones, contactos/calendario, movimiento, fotos, SMS y actualización de aplicaciones

> **ℹ️** Las rutas heredadas para Claude, Codex, Gemini y Opencode han sido eliminadas. Pi es la única ruta de agente de codificación.
