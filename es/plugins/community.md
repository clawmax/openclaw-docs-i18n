# Plugins de la comunidad

Esta página rastrea **plugins de alta calidad mantenidos por la comunidad** para OpenClaw. Aceptamos PRs que agreguen plugins de la comunidad aquí cuando cumplen con el estándar de calidad.

## Requisitos para aparecer en la lista

-   El paquete del plugin está publicado en npmjs (instalable mediante `openclaw plugins install <npm-spec>`).
-   El código fuente está alojado en GitHub (repositorio público).
-   El repositorio incluye documentación de configuración/uso y un rastreador de incidencias.
-   El plugin tiene una señal clara de mantenimiento (mantenedor activo, actualizaciones recientes o manejo receptivo de incidencias).

## Cómo enviar

Abre un PR que agregue tu plugin a esta página con:

-   Nombre del plugin
-   Nombre del paquete npm
-   URL del repositorio de GitHub
-   Descripción de una línea
-   Comando de instalación

## Estándar de revisión

Preferimos plugins que sean útiles, estén documentados y sean seguros de operar. Se pueden rechazar envoltorios de bajo esfuerzo, propiedad poco clara o paquetes sin mantenimiento.

## Formato para candidatos

Usa este formato al agregar entradas:

-   **Nombre del Plugin** — descripción breve npm: `@scope/paquete` repo: `https://github.com/org/repo` instalar: `openclaw plugins install @scope/paquete`
