# Плагины сообщества

На этой странице отслеживаются качественные **плагины, поддерживаемые сообществом** для OpenClaw. Мы принимаем PR, добавляющие сюда плагины сообщества, когда они соответствуют требуемому уровню качества.

## Требования для включения в список

-   Пакет плагина опубликован на npmjs (устанавливается через `openclaw plugins install <npm-spec>`).
-   Исходный код размещён на GitHub (публичный репозиторий).
-   Репозиторий включает документацию по настройке/использованию и трекер задач.
-   Плагин имеет чёткий сигнал о поддержке (активный сопровождающий, недавние обновления или оперативная работа с проблемами).

## Как добавить плагин

Откройте PR, который добавляет ваш плагин на эту страницу, указав:

-   Название плагина
-   Имя пакета в npm
-   URL репозитория на GitHub
-   Краткое описание в одну строку
-   Команду установки

## Критерии проверки

Мы отдаём предпочтение плагинам, которые полезны, задокументированы и безопасны в эксплуатации. Простые обёртки, плагины с неясным владельцем или неподдерживаемые пакеты могут быть отклонены.

## Формат записи

Используйте следующий формат при добавлении записей:

-   **Название плагина** — краткое описание npm: `@scope/package` репозиторий: `https://github.com/org/repo` установка: `openclaw plugins install @scope/package`

## Список плагинов

-   **WeChat** — Подключение OpenClaw к личным аккаунтам WeChat через WeChatPadPro (протокол iPad). Поддерживает обмен текстом, изображениями и файлами с беседами, запускаемыми по ключевым словам. npm: `@icesword760/openclaw-wechat` репозиторий: `https://github.com/icesword0760/openclaw-wechat` установка: `openclaw plugins install @icesword760/openclaw-wechat`
