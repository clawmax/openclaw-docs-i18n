## Что такое OpenClaw?

OpenClaw — это **саморазмещаемый шлюз**, который соединяет ваши любимые мессенджеры — WhatsApp, Telegram, Discord, iMessage и другие — с ИИ-агентами для программирования, такими как Pi. Вы запускаете один процесс Шлюза на своём компьютере (или сервере), и он становится мостом между вашими мессенджерами и всегда доступным ИИ-ассистентом.

**Для кого это?** Для разработчиков и опытных пользователей, которые хотят иметь личного ИИ-ассистента, которому можно писать откуда угодно — без передачи контроля над своими данными или зависимости от хостинговых сервисов.

**Что отличает его?**

- **Саморазмещение**: работает на вашем оборудовании, по вашим правилам
- **Многоканальность**: один Шлюз обслуживает WhatsApp, Telegram, Discord и другие одновременно
- **Нативная работа с агентами**: создан для ИИ-агентов с использованием инструментов, сессиями, памятью и маршрутизацией между несколькими агентами
- **Открытый исходный код**: лицензия MIT, развивается сообществом

**Что вам нужно?** Node 22+, API-ключ от выбранного провайдера и 5 минут. Для лучшего качества и безопасности используйте самую мощную модель последнего поколения.

## Как это работает

Шлюз является единым источником истины для сессий, маршрутизации и подключений к каналам.

## Ключевые возможности

  - **Многоканальный шлюз** — WhatsApp, Telegram, Discord и iMessage в рамках одного процесса Шлюза.

  - **Каналы через плагины** — Добавляйте Mattermost и другие с помощью пакетов расширений.

  - **Маршрутизация между агентами** — Изолированные сессии для каждого агента, рабочего пространства или отправителя.

  - **Поддержка медиа** — Отправляйте и получайте изображения, аудио и документы.

  - **Веб-панель управления** — Браузерная панель для чата, конфигурации, сессий и узлов.

  

## Быстрый старт

  ### Установите OpenClaw

```bash
npm install -g openclaw@latest
```

  

  ### Подключитесь и установите службу

```bash
openclaw onboard --install-daemon
```

  

  ### Сопрягите WhatsApp и запустите Шлюз

```bash
openclaw channels login
openclaw gateway --port 18789
```

  

Нужна полная установка и настройка для разработки? См. [Быстрый старт](./start/quickstart.md).

## Панель управления

Откройте браузерную Панель управления после запуска Шлюза.

- По умолчанию локально: [http://127.0.0.1:18789/](http://127.0.0.1:18789/)
- Удалённый доступ: [Веб-интерфейсы](./web.md) и [Tailscale](./gateway/tailscale.md)

## Конфигурация (опционально)

Конфигурация находится в файле `~/.openclaw/openclaw.json`.

- Если вы **ничего не делаете**, OpenClaw использует встроенный бинарный файл Pi в режиме RPC с сессиями для каждого отправителя.
- Если хотите усилить безопасность, начните с `channels.whatsapp.allowFrom` и (для групп) правил упоминаний.

Пример:

```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"],
      "groups": { "*": { "requireMention": true } }
    }
  },
  "messages": { "groupChat": { "mentionPatterns": ["@openclaw"] } }
}
```
