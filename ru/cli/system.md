  Команды CLI

  
# system

Системные вспомогательные средства для шлюза: постановка системных событий в очередь, управление пульсом и просмотр присутствия.

## Часто используемые команды

```bash
openclaw system event --text "Check for urgent follow-ups" --mode now
openclaw system heartbeat enable
openclaw system heartbeat last
openclaw system presence
```

## system event

Поставить системное событие в очередь для **основной** сессии. Следующий пульс внедрит его как строку `System:` в промпт. Используйте `--mode now` для немедленного запуска пульса; `next-heartbeat` ожидает следующего запланированного такта. Флаги:

-   `--text <текст>`: обязательный текст системного события.
-   `--mode <режим>`: `now` или `next-heartbeat` (по умолчанию).
-   `--json`: машинно-читаемый вывод.

## system heartbeat last|enable|disable

Управление пульсом:

-   `last`: показать последнее событие пульса.
-   `enable`: снова включить пульс (используйте, если он был отключен).
-   `disable`: приостановить пульс.

Флаги:

-   `--json`: машинно-читаемый вывод.

## system presence

Вывести список текущих записей о системном присутствии, известных шлюзу (узлы, экземпляры и аналогичные строки состояния). Флаги:

-   `--json`: машинно-читаемый вывод.

## Примечания

-   Требуется работающий шлюз, доступный по текущей конфигурации (локальный или удаленный).
-   Системные события являются временными и не сохраняются после перезапусков.
