# skills

Проверяйте навыки (встроенные + рабочие + управляемые переопределения) и смотрите, какие соответствуют требованиям, а каким чего-то не хватает. Связанные разделы:

-   Система навыков: [Навыки](../tools/skills.md)
-   Конфигурация навыков: [Конфигурация навыков](../tools/skills-config.md)
-   Установки ClawHub: [ClawHub](../tools/clawhub.md)

## Команды

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
