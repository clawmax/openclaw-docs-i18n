# health

Получение данных о состоянии работающего Шлюза.

```bash
openclaw health
openclaw health --json
openclaw health --verbose
```

Примечания:

-   `--verbose` запускает живые проверки и выводит время выполнения для каждого аккаунта, когда настроено несколько аккаунтов.
-   Вывод включает хранилища сессий для каждого агента, когда настроено несколько агентов.
