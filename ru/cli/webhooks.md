Помощники вебхуков и интеграции (Gmail Pub/Sub, помощники вебхуков). Связанные темы:

## Gmail

```bash
openclaw webhooks gmail setup --account you@example.com
openclaw webhooks gmail run
```

Подробности смотрите в [документации по Gmail Pub/Sub](../automation/gmail-pubsub.md).
