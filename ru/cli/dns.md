# dns

DNS-помощники для широковещательного обнаружения (Tailscale + CoreDNS). В настоящее время ориентированы на macOS + Homebrew CoreDNS. Связанные темы:

-   Обнаружение шлюза: [Discovery](../gateway/discovery.md)
-   Конфигурация широковещательного обнаружения: [Configuration](../gateway/configuration.md)

## Настройка

```bash
openclaw dns setup
openclaw dns setup --apply
```
