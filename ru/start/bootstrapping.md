# Начальная загрузка агента

Начальная загрузка — это **ритуал первого запуска**, который подготавливает рабочую область агента и собирает сведения об идентичности. Это происходит после онбординга, когда агент запускается впервые.

## Что делает начальная загрузка

При первом запуске агента OpenClaw выполняет начальную загрузку рабочей области (по умолчанию `~/.openclaw/workspace`):

-   Создаёт начальные файлы `AGENTS.md`, `BOOTSTRAP.md`, `IDENTITY.md`, `USER.md`.
-   Запускает короткий ритуал вопросов и ответов (по одному вопросу за раз).
-   Записывает данные об идентичности и предпочтениях в `IDENTITY.md`, `USER.md`, `SOUL.md`.
-   Удаляет `BOOTSTRAP.md` по завершении, чтобы процесс выполнялся только один раз.

## Где это выполняется

Начальная загрузка всегда выполняется на **хосте-шлюзе**. Если приложение macOS подключается к удалённому Шлюзу, рабочая область и файлы начальной загрузки находятся на этой удалённой машине.

> **ℹ️** Если Шлюз работает на другой машине, редактируйте файлы рабочей области на хосте-шлюзе (например, `user@gateway-host:~/.openclaw/workspace`).
