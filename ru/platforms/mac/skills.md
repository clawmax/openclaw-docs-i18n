# Навыки

Приложение для macOS отображает навыки OpenClaw через шлюз; оно не обрабатывает навыки локально.

## Источник данных

-   `skills.status` (шлюз) возвращает все навыки, а также информацию об их доступности и отсутствующих требованиях (включая блокировки по списку разрешений для встроенных навыков).
-   Требования определяются из `metadata.openclaw.requires` в каждом файле `SKILL.md`.

## Действия по установке

-   `metadata.openclaw.install` определяет варианты установки (brew/node/go/uv).
-   Приложение вызывает `skills.install` для запуска установщиков на хосте шлюза.
-   Шлюз предоставляет только один предпочтительный установщик, когда доступно несколько (brew, если доступен, иначе менеджер node из `skills.install`, по умолчанию npm).

## Переменные окружения/API-ключи

-   Приложение хранит ключи в `~/.openclaw/openclaw.json` в разделе `skills.entries.`.
-   `skills.update` обновляет параметры `enabled`, `apiKey` и `env`.

## Удаленный режим

-   Установка и обновление конфигурации происходят на хосте шлюза (а не на локальном Mac).
