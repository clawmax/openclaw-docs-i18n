# AGENTS.md по умолчанию

## Первый запуск (рекомендуется)

OpenClaw использует выделенный каталог рабочего пространства для агента. По умолчанию: `~/.openclaw/workspace` (настраивается через `agents.defaults.workspace`).

1.  Создайте рабочее пространство (если оно еще не существует):

```bash
mkdir -p ~/.openclaw/workspace
```

2.  Скопируйте шаблоны рабочего пространства по умолчанию в рабочее пространство:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3.  Опционально: если вам нужен список навыков персонального помощника, замените AGENTS.md этим файлом:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4.  Опционально: выберите другое рабочее пространство, установив `agents.defaults.workspace` (поддерживает `~`):

```json
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## Настройки безопасности по умолчанию

-   Не выгружайте каталоги или секреты в чат.
-   Не выполняйте деструктивные команды, если об этом явно не попросили.
-   Не отправляйте частичные/потоковые ответы во внешние мессенджеры (только финальные ответы).

## Начало сессии (обязательно)

-   Прочтите `SOUL.md`, `USER.md`, `memory.md`, а также записи за сегодня и вчера в `memory/`.
-   Сделайте это перед ответом.

## Душа (обязательно)

-   `SOUL.md` определяет личность, тон и границы. Поддерживайте его в актуальном состоянии.
-   Если вы изменили `SOUL.md`, сообщите об этом пользователю.
-   Вы — новый экземпляр в каждой сессии; преемственность сохраняется в этих файлах.

## Общие пространства (рекомендуется)

-   Вы не являетесь голосом пользователя; будьте осторожны в групповых чатах или публичных каналах.
-   Не делитесь приватными данными, контактной информацией или внутренними заметками.

## Система памяти (рекомендуется)

-   Ежедневный журнал: `memory/YYYY-MM-DD.md` (создайте `memory/`, если нужно).
-   Долговременная память: `memory.md` для постоянных фактов, предпочтений и решений.
-   При запуске сессии прочтите записи за сегодня, вчера и `memory.md`, если они есть.
-   Записывайте: решения, предпочтения, ограничения, незавершенные задачи.
-   Избегайте секретов, если об этом явно не попросили.

## Инструменты и навыки

-   Инструменты находятся в навыках; следуйте `SKILL.md` каждого навыка, когда он вам нужен.
-   Храните заметки, специфичные для окружения, в `TOOLS.md` (Заметки для Навыков).

## Совет по резервному копированию (рекомендуется)

Если вы рассматриваете это рабочее пространство как «память» Clawd, сделайте его git-репозиторием (желательно приватным), чтобы `AGENTS.md` и ваши файлы памяти были сохранены.

```bash
cd ~/.openclaw/workspace
git init
git add AGENTS.md
git commit -m "Add Clawd workspace"
# Опционально: добавьте приватный удаленный репозиторий и выполните push
```

## Что делает OpenClaw

-   Запускает шлюз WhatsApp + агента для написания кода на Pi, чтобы помощник мог читать/писать чаты, получать контекст и запускать навыки через хост-компьютер Mac.
-   Приложение macOS управляет разрешениями (запись экрана, уведомления, микрофон) и предоставляет доступ к CLI `openclaw` через встроенный бинарный файл.
-   Прямые чаты по умолчанию объединяются в сессию `main` агента; группы остаются изолированными как `agent:::group:` (комнаты/каналы: `agent:::channel:`); пульсации (heartbeats) поддерживают фоновые задачи активными.

## Основные навыки (включите в Настройки → Навыки)

-   **mcporter** — Сервер инструментов/CLI для управления внешними бэкендами навыков.
-   **Peekaboo** — Быстрые скриншоты macOS с опциональным анализом зрения ИИ.
-   **camsnap** — Захват кадров, клипов или оповещений о движении с камер безопасности RTSP/ONVIF.
-   **oracle** — Готовый к работе с OpenAI CLI агента с воспроизведением сессии и управлением браузером.
-   **eightctl** — Управляйте своим сном из терминала.
-   **imsg** — Отправляйте, читайте, транслируйте iMessage и SMS.
-   **wacli** — WhatsApp CLI: синхронизация, поиск, отправка.
-   **discord** — Действия в Discord: реакции, стикеры, опросы. Используйте цели `user:` или `channel:` (простые числовые идентификаторы неоднозначны).
-   **gog** — Google Suite CLI: Gmail, Календарь, Диск, Контакты.
-   **spotify-player** — Терминальный клиент Spotify для поиска/очереди/управления воспроизведением.
-   **sag** — Речь ElevenLabs с UX, похожим на mac `say`; по умолчанию транслирует на динамики.
-   **Sonos CLI** — Управление динамиками Sonos (обнаружение/статус/воспроизведение/громкость/группировка) из скриптов.
-   **blucli** — Воспроизведение, группировка и автоматизация плееров BluOS из скриптов.
-   **OpenHue CLI** — Управление освещением Philips Hue для сцен и автоматизации.
-   **OpenAI Whisper** — Локальное преобразование речи в текст для быстрой диктовки и расшифровки голосовой почты.
-   **Gemini CLI** — Модели Google Gemini из терминала для быстрых вопросов и ответов.
-   **agent-tools** — Набор утилит для автоматизации и вспомогательных скриптов.

## Примечания по использованию

-   Предпочитайте CLI `openclaw` для скриптов; приложение mac обрабатывает разрешения.
-   Запускайте установку из вкладки Навыки; кнопка скрывается, если бинарный файл уже присутствует.
-   Поддерживайте пульсации (heartbeats) включенными, чтобы помощник мог планировать напоминания, отслеживать входящие и запускать захват с камер.
-   Интерфейс Canvas работает в полноэкранном режиме с нативными оверлеями. Избегайте размещения критически важных элементов управления в верхнем левом/правом углу или нижнем краю; добавляйте явные отступы в макете и не полагайтесь на безопасные области (safe-area insets).
-   Для проверки через браузер используйте `openclaw browser` (вкладки/статус/скриншот) с управляемым OpenClaw профилем Chrome.
-   Для инспекции DOM используйте `openclaw browser eval|query|dom|snapshot` (и `--json`/`--out`, когда нужен машинный вывод).
-   Для взаимодействий используйте `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` (click/type требуют ссылок на снимок; используйте `evaluate` для CSS-селекторов).
