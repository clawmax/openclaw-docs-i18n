# Шаблон TOOLS.md

Навыки определяют *как* работают инструменты. Этот файл предназначен для *ваших* конкретных данных — того, что уникально для вашей настройки.

## Что сюда входит

Такие вещи, как:

-   Названия и расположение камер
-   SSH-хосты и алиасы
-   Предпочитаемые голоса для TTS
-   Названия колонок/комнат
-   Прозвища устройств
-   Любые специфичные для окружения данные

## Примеры

```markdown
### Камеры

- living-room → Основная зона, широкоугольная 180°
- front-door → Вход, с датчиком движения

### SSH

- home-server → 192.168.1.100, пользователь: admin

### TTS

- Предпочитаемый голос: "Nova" (тёплый, слегка британский)
- Колонка по умолчанию: Кухонный HomePod
```

## Почему отдельно?

Навыки являются общими. Ваша настройка — ваша. Разделение их означает, что вы можете обновлять навыки, не теряя своих заметок, и делиться навыками, не раскрывая свою инфраструктуру.
