# Модели GLM

GLM — это **семейство моделей** (не компания), доступное через платформу Z.AI. В OpenClaw модели GLM доступны через провайдера `zai` и идентификаторы моделей, такие как `zai/glm-5`.

## Настройка CLI

```bash
openclaw onboard --auth-choice zai-api-key
```

## Фрагмент конфигурации

```json
{
  env: { ZAI_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "zai/glm-5" } } },
}
```

## Примечания

-   Версии и доступность моделей GLM могут меняться; проверяйте актуальную информацию в документации Z.AI.
-   Примеры идентификаторов моделей: `glm-5`, `glm-4.7` и `glm-4.6`.
-   Подробности о провайдере см. на странице [/providers/zai](./zai.md).
