# Возможности

## Основные моменты

## Каналы

WhatsApp, Telegram, Discord и iMessage через единый Шлюз.

## Плагины

Добавляйте Mattermost и другие системы с помощью расширений.

## Маршрутизация

Маршрутизация между несколькими агентами с изолированными сессиями.

## Медиа

Изображения, аудио и документы на вход и выход.

## Приложения и интерфейс

Веб-интерфейс управления и приложение-компаньон для macOS.

## Мобильные узлы

Узлы для iOS и Android с сопряжением, голосом/чатом и расширенными командами для устройств.

## Полный список

-   Интеграция WhatsApp через WhatsApp Web (Baileys)
-   Поддержка ботов Telegram (grammY)
-   Поддержка ботов Discord (channels.discord.js)
-   Поддержка ботов Mattermost (плагин)
-   Интеграция iMessage через локальный CLI imsg (macOS)
-   Агентский мост для Pi в режиме RPC с потоковой передачей инструментов
-   Потоковая передача и разбиение на части для длинных ответов
-   Маршрутизация между несколькими агентами для изолированных сессий на рабочее пространство или отправителя
-   Авторизация по подписке для Anthropic и OpenAI через OAuth
-   Сессии: прямые чаты объединяются в общую `main`; группы изолированы
-   Поддержка групповых чатов с активацией по упоминанию
-   Поддержка медиа: изображения, аудио и документы
-   Опциональный перехватчик для транскрибации голосовых сообщений
-   WebChat и приложение для строки меню macOS
-   Узел iOS с сопряжением, Canvas, камерой, записью экрана, геолокацией и голосовыми функциями
-   Узел Android с сопряжением, вкладкой Connect, сессиями чата, голосовой вкладкой, Canvas/камерой/экраном, а также командами для устройства, уведомлений, контактов/календаря, движения, фотографий, SMS и обновления приложений

> **ℹ️** Устаревшие пути Claude, Codex, Gemini и Opencode были удалены. Pi — единственный путь для агента-кодера.
