# Отслеживание использования

## Что это такое

-   Получает данные об использовании/квотах провайдера напрямую с их конечных точек использования.
-   Без примерных затрат; только окна, о которых сообщает провайдер.

## Где отображается

-   `/status` в чатах: карточка статуса с эмодзи, содержащая токены сессии + примерная стоимость (только для API-ключа). Использование провайдера показывается для **текущего провайдера моделей**, когда доступно.
-   `/usage off|tokens|full` в чатах: футер с использованием на каждый ответ (OAuth показывает только токены).
-   `/usage cost` в чатах: локальная сводка затрат, агрегированная из логов сессий OpenClaw.
-   CLI: `openclaw status --usage` выводит полную детализацию по каждому провайдеру.
-   CLI: `openclaw channels list` выводит тот же снимок использования вместе с конфигурацией провайдера (используйте `--no-usage`, чтобы пропустить).
-   Меню-бар macOS: раздел «Usage» (Использование) в Контексте (только если доступно).

## Провайдеры + учетные данные

-   **Anthropic (Claude)**: OAuth-токены в профилях аутентификации.
-   **GitHub Copilot**: OAuth-токены в профилях аутентификации.
-   **Gemini CLI**: OAuth-токены в профилях аутентификации.
-   **Antigravity**: OAuth-токены в профилях аутентификации.
-   **OpenAI Codex**: OAuth-токены в профилях аутентификации (используется accountId, если присутствует).
-   **MiniMax**: API-ключ (ключ кодирующего плана; `MINIMAX_CODE_PLAN_KEY` или `MINIMAX_API_KEY`); использует 5‑часовое окно кодирующего плана.
-   **z.ai**: API-ключ через env/config/auth store.

Использование скрыто, если нет соответствующих учетных данных OAuth/API.
