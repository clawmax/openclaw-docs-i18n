# setup

`~/.openclaw/openclaw.json`とエージェントワークスペースを初期化します。関連項目:

-   はじめに: [はじめに](../start/getting-started.md)
-   ウィザード: [オンボーディング](../start/onboarding.md)

## 例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

セットアップ経由でウィザードを実行するには:

```bash
openclaw setup --wizard
```
