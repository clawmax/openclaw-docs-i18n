# skills

スキル（バンドル済み + ワークスペース + 管理上書き）を検査し、適格なものと不足要件を確認します。関連項目:

-   スキルシステム: [スキル](../tools/skills.md)
-   スキル設定: [スキル設定](../tools/skills-config.md)
-   ClawHubインストール: [ClawHub](../tools/clawhub.md)

## コマンド

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
