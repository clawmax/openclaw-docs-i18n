# dns

広域ディスカバリー（Tailscale + CoreDNS）のためのDNSヘルパー。現在はmacOS + Homebrew CoreDNSに焦点を当てています。関連項目:

-   ゲートウェイディスカバリー: [ディスカバリー](../gateway/discovery.md)
-   広域ディスカバリー設定: [設定](../gateway/configuration.md)

## セットアップ

```bash
openclaw dns setup
openclaw dns setup --apply
```
