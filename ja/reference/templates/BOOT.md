# BOOT.md テンプレート

OpenClaw が起動時に実行すべき短く明確な指示を追加します（`hooks.internal.enabled` を有効化）。タスクがメッセージを送信する場合は、メッセージツールを使用し、その後 NO_REPLY で応答します。
