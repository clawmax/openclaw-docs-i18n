# TOOLS.md テンプレート

スキルはツールの*動作方法*を定義します。このファイルは、あなたのセットアップに固有の詳細、つまり*あなただけのもの*を記述するためのものです。

## ここに記述する内容

以下のようなもの:

-   カメラの名前と設置場所
-   SSHホストとエイリアス
-   TTS用の好みの音声
-   スピーカー/部屋の名前
-   デバイスの愛称
-   環境固有のあらゆるもの

## 例

```markdown
### カメラ

- living-room → メインエリア、180°広角
- front-door → 玄関、モーション検知式

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 好みの音声: "Nova" (温かみのある、少し英国風)
- デフォルトスピーカー: キッチンのHomePod
```

## なぜ分けるのか？

スキルは共有されます。あなたのセットアップはあなただけのものです。これらを分けておくことで、あなたのメモを失うことなくスキルを更新でき、あなたのインフラ情報を漏らすことなくスキルを共有できます。
