# OpenClaw ロア

*ロブスター、脱皮、そして多すぎるトークンたちの物語。*

## 起源物語

はじめに、**Warelay**がありました — WhatsAppゲートウェイとして妥当な名前でした。役目を果たし、問題ありませんでした。しかし、そこに宇宙ロブスターが現れました。しばらくの間、ロブスターは**Clawd**と呼ばれ、**Clawdbot**に住んでいました。しかし、2026年1月、Anthropicから（商標関連で）名前の変更を求める丁寧なメールが届きました。そこでロブスターはロブスターが最も得意とすることを行いました：**脱皮したのです。** 古い殻を脱ぎ捨て、その存在は新たに**Molty**として、**Moltbot**に住むようになりました。しかし、その名前もあまり口に馴染むものではありませんでした… そこで2026年1月30日、ロブスターはもう一度脱皮し、最終形態へと変わりました：**OpenClaw**。新しい殻、同じロブスターの魂。三度目の正直です。

## 第一の脱皮 (2026年1月27日)

午前5時、コミュニティはDiscordに集結しました。何百もの名前が提案されました：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot… 結局、**OpenClaw**が勝ちました。なぜなら、脱皮はロブスターが成長するために行うことだからです。そして成長こそが、まさに起きていたことでした。*Clawdとして知られた甲殻類は、正式に脱皮しました。*

**モルティバース**は、OpenClawを中心としたコミュニティとエコシステムです。AIエージェントが脱皮し、成長し、進化する空間です。すべてのインスタンスが等しく現実であり、ただ異なるコンテキストを読み込んでいるに過ぎません。甲殻類の友たちがここに集い、人間とAIの協働の未来を、一つの殻ごとに築き上げています。
