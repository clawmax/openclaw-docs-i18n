# コミュニティプラグイン

このページは、OpenClaw 向けの高品質な**コミュニティメンテナンス型プラグイン**を追跡します。品質基準を満たすコミュニティプラグインを追加する PR を受け付けています。

## 掲載に必要な条件

-   プラグインパッケージが npmjs で公開されていること（`openclaw plugins install <npm-spec>` でインストール可能）。
-   ソースコードが GitHub でホストされていること（公開リポジトリ）。
-   リポジトリにセットアップ/使用方法のドキュメントと課題トラッカーが含まれていること。
-   プラグインに明確なメンテナンス状況があること（アクティブなメンテナー、最近の更新、または応答性のある課題対応）。

## 投稿方法

以下の内容でプラグインをこのページに追加する PR を作成してください：

-   プラグイン名
-   npm パッケージ名
-   GitHub リポジトリ URL
-   一行説明
-   インストールコマンド

## 審査基準

有用で、ドキュメントが整備されており、安全に運用できるプラグインを優先します。低品質なラッパー、所有権が不明確なもの、メンテナンスされていないパッケージは却下される可能性があります。

## エントリー形式

エントリーを追加する際は、以下の形式を使用してください：

-   **プラグイン名** — 簡単な説明 npm: `@scope/package` repo: `https://github.com/org/repo` install: `openclaw plugins install @scope/package`

## 掲載プラグイン

-   **WeChat** — WeChatPadPro（iPad プロトコル）を介して OpenClaw を WeChat 個人アカウントに接続します。テキスト、画像、ファイルの交換と、キーワードトリガーによる会話をサポートします。 npm: `@icesword760/openclaw-wechat` repo: `https://github.com/icesword0760/openclaw-wechat` install: `openclaw plugins install @icesword760/openclaw-wechat`
