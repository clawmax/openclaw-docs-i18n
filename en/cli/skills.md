# skills

Inspect skills (bundled + workspace + managed overrides) and see what’s eligible vs missing requirements. Related:

-   Skills system: [Skills](../tools/skills.md)
-   Skills config: [Skills config](../tools/skills-config.md)
-   ClawHub installs: [ClawHub](../tools/clawhub.md)

## Commands

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
