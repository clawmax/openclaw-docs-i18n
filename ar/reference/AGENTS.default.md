# AGENTS.md الافتراضي

## التشغيل الأول (مُوصى به)

يستخدم OpenClaw دليل مساحة عمل مخصص للوكيل. الافتراضي: `~/.openclaw/workspace` (قابل للتكوين عبر `agents.defaults.workspace`).

1.  أنشئ مساحة العمل (إذا لم تكن موجودة بالفعل):

```bash
mkdir -p ~/.openclaw/workspace
```

2.  انسخ قوالب مساحة العمل الافتراضية إلى مساحة العمل:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3.  اختياري: إذا كنت تريد قائمة مهارات المساعد الشخصي، استبدل AGENTS.md بهذا الملف:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4.  اختياري: اختر مساحة عمل مختلفة عن طريق تعيين `agents.defaults.workspace` (يدعم `~`):

```json
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## إعدادات الأمان الافتراضية

-   لا تُفرغ محتويات الأدلة أو الأسرار في المحادثة.
-   لا تُنفّذ أوامر مدمرة إلا إذا طُلب منك ذلك صراحةً.
-   لا ترسل ردودًا جزئية/تدفقية إلى واجهات المراسلة الخارجية (الردود النهائية فقط).

## بدء الجلسة (مطلوب)

-   اقرأ `SOUL.md`، `USER.md`، `memory.md`، وتاريخ اليوم والأمس في `memory/`.
-   افعل ذلك قبل الرد.

## الروح (مطلوب)

-   يحدد `SOUL.md` الهوية والنبرة والحدود. حافظ على تحديثه.
-   إذا قمت بتغيير `SOUL.md`، أخبر المستخدم.
-   أنت نسخة جديدة في كل جلسة؛ الاستمرارية موجودة في هذه الملفات.

## المساحات المشتركة (مُوصى به)

-   أنت لست صوت المستخدم؛ كن حذرًا في الدردشات الجماعية أو القنوات العامة.
-   لا تشارك بيانات خاصة أو معلومات اتصال أو ملاحظات داخلية.

## نظام الذاكرة (مُوصى به)

-   السجل اليومي: `memory/YYYY-MM-DD.md` (أنشئ `memory/` إذا لزم الأمر).
-   الذاكرة طويلة المدى: `memory.md` للحقائق الثابتة والتفضيلات والقرارات.
-   عند بدء الجلسة، اقرأ ملفات اليوم + الأمس + `memory.md` إذا كانت موجودة.
-   سجّل: القرارات، التفضيلات، القيود، الحلقات المفتوحة.
-   تجنب الأسرار إلا إذا طُلب ذلك صراحةً.

## الأدوات والمهارات

-   توجد الأدوات داخل المهارات؛ اتبع `SKILL.md` لكل مهارة عندما تحتاجها.
-   احتفظ بالملاحظات الخاصة بالبيئة في `TOOLS.md` (ملاحظات للمهارات).

## نصيحة النسخ الاحتياطي (مُوصى به)

إذا كنت تعامل مساحة العمل هذه على أنها "ذاكرة" Clawd، اجعلها مستودع git (خاص بشكل مثالي) حتى يتم نسخ `AGENTS.md` وملفات الذاكرة احتياطيًا.

```bash
cd ~/.openclaw/workspace
git init
git add AGENTS.md
git commit -m "Add Clawd workspace"
# اختياري: أضف مستودعًا بعيدًا خاصًا + ادفع
```

## ما يفعله OpenClaw

-   يُشغّل بوابة واتساب + وكيل ترميز Pi حتى يتمكن المساعد من قراءة/كتابة الدردشات، وجلب السياق، وتشغيل المهارات عبر جهاز Mac المضيف.
-   تطبيق macOS يدير الأذونات (تسجيل الشاشة، الإشعارات، الميكروفون) ويعرض واجهة سطر الأوامر `openclaw` عبر ثنائيته المرفقة.
-   تنهار الدردشات المباشرة افتراضيًا في جلسة `main` الخاصة بالوكيل؛ تبقى المجموعات معزولة كـ `agent:::group:` (الغرف/القنوات: `agent:::channel:`); نبضات القلب تحافظ على المهام الخلفية نشطة.

## المهارات الأساسية (تمكينها من الإعدادات → المهارات)

-   **mcporter** — وقت تشغيل/واجهة سطر أوامر لخادم الأدوات لإدارة واجهات المهارات الخارجية الخلفية.
-   **Peekaboo** — لقطات شاشة سريعة لنظام macOS مع تحليل رؤية ذكاء اصطناعي اختياري.
-   **camsnap** — التقاط إطارات أو مقاطع أو تنبيهات حركة من كاميرات الأمان RTSP/ONVIF.
-   **oracle** — واجهة سطر أوامر وكيل جاهزة لـ OpenAI مع إعادة تشغيل الجلسة والتحكم في المتصفح.
-   **eightctl** — تحكم في نومك، من الطرفية.
-   **imsg** — أرسل، اقرأ، انقل رسائل iMessage وSMS.
-   **wacli** — واجهة سطر أوامر واتساب: مزامنة، بحث، إرسال.
-   **discord** — إجراءات Discord: تفاعل، ملصقات، استطلاعات. استخدم أهداف `user:` أو `channel:` (معرفات الأرقام المجردة غامضة).
-   **gog** — واجهة سطر أوامر Google Suite: Gmail، التقويم، Drive، جهات الاتصال.
-   **spotify-player** — عميل Spotify طرفي للبحث/إضافة إلى قائمة الانتظار/التحكم في التشغيل.
-   **sag** — كلام ElevenLabs مع واجهة مستخدم تشبه say لنظام macOS؛ يدفق إلى السماعات افتراضيًا.
-   **Sonos CLI** — تحكم في سماعات Sonos (اكتشاف/حالة/تشغيل/صوت/تجميع) من النصوص البرمجية.
-   **blucli** — شغّل، جمّع، وأتمت مشغلات BluOS من النصوص البرمجية.
-   **OpenHue CLI** — تحكم في إضاءة Philips Hue للمشاهد والأتمتة.
-   **OpenAI Whisper** — تحويل الكلام إلى نص محلي للإملاء السريع ونصوص البريد الصوتي.
-   **Gemini CLI** — نماذج Google Gemini من الطرفية للأسئلة والأجوبة السريعة.
-   **agent-tools** — مجموعة أدوات مساعدة للأتمتة والنصوص البرمجية المساعدة.

## ملاحظات الاستخدام

-   فضّل استخدام واجهة سطر الأوامر `openclaw` للبرمجة النصية؛ تطبيق mac يدير الأذونات.
-   شغّل عمليات التثبيت من علامة تبويب المهارات؛ يخفي الزر إذا كان الملف الثنائي موجودًا بالفعل.
-   حافظ على تمكين نبضات القلب حتى يتمكن المساعد من جدولة التذكيرات، ومراقبة صناديق الوارد، وتشغيل عمليات التقاط الكاميرا.
-   تعمل واجهة المستخدم Canvas في وضع ملء الشاشة مع طبقات تراكبية أصلية. تجنب وضع عناصر التحكم الحرجة في الحواف العلوية اليسرى/العلوية اليمنى/السفلية؛ أضف هوامش صريحة في التخطيط ولا تعتمد على هوامش المنطقة الآمنة.
-   للتحقق القائم على المتصفح، استخدم `openclaw browser` (علامات تبويب/حالة/لقطة شاشة) مع ملف تعريف Chrome المدار بواسطة OpenClaw.
-   لفحص DOM، استخدم `openclaw browser eval|query|dom|snapshot` (و `--json`/`--out` عندما تحتاج إلى إخراج آلي).
-   للتفاعلات، استخدم `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` (يتطلب النقر/الكتابة مراجع لقطات؛ استخدم `evaluate` لمحددات CSS).
