# قالب TOOLS.md

المهارات تحدد *كيفية* عمل الأدوات. هذا الملف مخصص لتفاصيلك *الخاصة* — الأشياء الفريدة في إعدادك.

## ما الذي يوضع هنا

أشياء مثل:

-   أسماء الكاميرات ومواقعها
-   خوادم SSH وأسمائها المستعارة
-   الأصوات المفضلة لتحويل النص إلى كلام
-   أسماء السماعات/الغرف
-   أسماء الأجهزة المستعارة
-   أي شيء خاص بالبيئة

## أمثلة

```markdown
### الكاميرات

- living-room → المنطقة الرئيسية، زاوية واسعة 180°
- front-door → المدخل، يتم تشغيلها بالحركة

### SSH

- home-server → 192.168.1.100، مستخدم: admin

### تحويل النص إلى كلام

- الصوت المفضل: "Nova" (دافئ، بريطاني قليلاً)
- السماعة الافتراضية: HomePod المطبخ
```

## لماذا الفصل؟

المهارات يتم مشاركتها. إعدادك هو ملكك. الفصل بينهما يعني أنه يمكنك تحديث المهارات دون فقدان ملاحظاتك، ومشاركة المهارات دون تسريب تفاصيل بنيتك التحتية.
