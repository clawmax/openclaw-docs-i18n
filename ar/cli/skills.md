# skills

افحص المهارات (المجمّعة + في مساحة العمل + التجاوزات المدارة) وشاهد ما هو مؤهل مقابل المتطلبات المفقودة. متعلق:

-   نظام المهارات: [المهارات](../tools/skills.md)
-   تهيئة المهارات: [تهيئة المهارات](../tools/skills-config.md)
-   تثبيتات ClawHub: [ClawHub](../tools/clawhub.md)

## الأوامر

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
