# setup

قم بتهيئة `~/.openclaw/openclaw.json` ومساحة عمل الوكيل. ذات صلة:

-   البدء: [البدء](../start/getting-started.md)
-   المعالج: [الترحيب](../start/onboarding.md)

## أمثلة

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

لتشغيل المعالج عبر الأمر setup:

```bash
openclaw setup --wizard
```
