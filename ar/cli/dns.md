# dns

مساعدات DNS لاكتشاف النطاق الواسع (Tailscale + CoreDNS). حالياً يركز على macOS + Homebrew CoreDNS. مواضيع ذات صلة:

-   اكتشاف البوابة: [Discovery](../gateway/discovery.md)
-   تكوين اكتشاف النطاق الواسع: [Configuration](../gateway/configuration.md)

## الإعداد

```bash
openclaw dns setup
openclaw dns setup --apply
```
