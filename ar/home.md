## ما هو OpenClaw؟

OpenClaw هي **بوابة ذاتية الاستضافة** تربط تطبيقات الدردشة المفضلة لديك — واتساب، تيليجرام، ديسكورد، آي ميسج، والمزيد — بوكلاء الذكاء الاصطناعي للبرمجة مثل Pi. تقوم بتشغيل عملية بوابة واحدة على جهازك الخاص (أو على خادم)، وتصبح الجسر بين تطبيقات المراسلة الخاصة بك ومساعد الذكاء الاصطناعي المتاح دائمًا.

**لمن صُممت؟** للمطورين والمستخدمين المتقدمين الذين يريدون مساعد ذكاء اصطناعي شخصي يمكنهم المراسلة منه من أي مكان — دون التخلي عن السيطرة على بياناتهم أو الاعتماد على خدمة مستضافة.

**ما الذي يميزها؟**

- **ذاتية الاستضافة**: تعمل على عتادك الخاص، وفق قواعدك
- **متعددة القنوات**: بوابة واحدة تخدم واتساب وتيليجرام وديسكورد والمزيد في وقت واحد
- **مصممة للوكلاء**: مبنية خصيصًا لوكلاء البرمجة مع استخدام الأدوات، الجلسات، الذاكرة، والتوجيه متعدد الوكلاء
- **مفتوحة المصدر**: مرخصة بـ MIT، وتقودها المجتمع

**ماذا تحتاج؟** Node 22+، مفتاح API من مزودك المختار، و5 دقائق. للحصول على أفضل جودة وأمان، استخدم أقوى نموذج من الجيل الأخير المتاح.

## كيف تعمل

البوابة هي المصدر الوحيد الموثوق للجلسات، والتوجيه، واتصالات القنوات.

## الميزات الرئيسية

  - **بوابة متعددة القنوات** — واتساب، تيليجرام، ديسكورد، وآي ميسج مع عملية بوابة واحدة.

  - **قنوات الإضافات** — أضف ماترموست والمزيد مع حزم التمديد.

  - **توجيه متعدد الوكلاء** — جلسات معزولة لكل وكيل، مساحة عمل، أو مرسل.

  - **دعم الوسائط** — أرسل واستقبل الصور، الصوت، والمستندات.

  - **واجهة تحكم الويب** — لوحة تحكم في المتصفح للدردشة، الإعدادات، الجلسات، والعُقد.

  

## البدء السريع

  ### تثبيت OpenClaw

```bash
npm install -g openclaw@latest
```

  

  ### الانضمام وتثبيت الخدمة

```bash
openclaw onboard --install-daemon
```

  

  ### اقتران واتساب وبدء تشغيل البوابة

```bash
openclaw channels login
openclaw gateway --port 18789
```

  

هل تحتاج إلى إعداد التثبيت والتطوير الكامل؟ راجع [البدء السريع](./start/quickstart.md).

## لوحة التحكم

افتح واجهة تحكم المتصفح بعد بدء تشغيل البوابة.

- المحلي افتراضيًا: [http://127.0.0.1:18789/](http://127.0.0.1:18789/)
- الوصول عن بُعد: [أسطح الويب](./web.md) و [Tailscale](./gateway/tailscale.md)

## الإعدادات (اختياري)

توجد الإعدادات في `~/.openclaw/openclaw.json`.

- إذا **لم تفعل شيئًا**، يستخدم OpenClaw ثنائي Pi المدمج في وضع RPC مع جلسات لكل مرسل.
- إذا أردت تأمينها، ابدأ بـ `channels.whatsapp.allowFrom` و (للمجموعات) قواعد الذكر.

مثال:

```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"],
      "groups": { "*": { "requireMention": true } }
    }
  },
  "messages": { "groupChat": { "mentionPatterns": ["@openclaw"] } }
}
```
