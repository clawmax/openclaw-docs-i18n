# إضافات المجتمع

تتتبع هذه الصفحة **الإضافات عالية الجودة التي يديرها المجتمع** لـ OpenClaw. نقبل طلبات السحب (PRs) التي تضيف إضافات مجتمعية هنا عندما تلبي معايير الجودة.

## المتطلبات للإدراج في القائمة

-   حزمة الإضافة منشورة على npmjs (قابلة للتثبيت عبر `openclaw plugins install <npm-spec>`).
-   الكود المصدري مستضاف على GitHub (مستودع عام).
-   يتضمن المستودع وثائق الإعداد/الاستخدام ونظام تتبع المشكلات.
-   للإضافة إشارة صيانة واضحة (مشرف نشط، تحديثات حديثة، أو معالجة مستجيبة للمشكلات).

## كيفية التقديم

افتح طلب سحب (PR) يضيف إضافتك إلى هذه الصفحة مع:

-   اسم الإضافة
-   اسم حزمة npm
-   رابط مستودع GitHub
-   وصف مختصر (سطر واحد)
-   أمر التثبيت

## معيار المراجعة

نفضل الإضافات المفيدة، الموثقة، والآلة في التشغيل. قد يتم رفض الالتفافات منخفضة الجهد، أو ملكية غير واضحة، أو الحزم غير المصرح بها.

## تنسيق المرشح

استخدم هذا التنسيق عند إضافة الإدخالات:

-   **اسم الإضافة** — وصف مختصر npm: `@scope/package` repo: `https://github.com/org/repo` install: `openclaw plugins install @scope/package`

## الإضافات المدرجة

-   **WeChat** — ربط OpenClaw بحسابات WeChat الشخصية عبر WeChatPadPro (بروتوكول iPad). يدعم تبادل النصوص والصور والملفات مع محادثات يتم تشغيلها بالكلمات المفتاحية. npm: `@icesword760/openclaw-wechat` repo: `https://github.com/icesword0760/openclaw-wechat` install: `openclaw plugins install @icesword760/openclaw-wechat`
