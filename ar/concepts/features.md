# الميزات

## أبرز المميزات

## القنوات

واتساب، تيليجرام، ديسكورد، و iMessage مع بوابة واحدة.

## الإضافات

أضف Mattermost والمزيد عبر الامتدادات.

## التوجيه

توجيه متعدد الوكلاء مع جلسات معزولة.

## الوسائط

الصور والصوت والمستندات داخليًا وخارجيًا.

## التطبيقات والواجهة

واجهة تحكم الويب وتطبيق macOS المصاحب.

## العقد المتنقلة

عقد iOS و Android مع الاقتران، الصوت/الدردشة، وأوامر الجهاز المتقدمة.

## القائمة الكاملة

-   تكامل واتساب عبر WhatsApp Web (Baileys)
-   دعم بوت تيليجرام (grammY)
-   دعم بوت ديسكورد (channels.discord.js)
-   دعم بوت Mattermost (إضافة)
-   تكامل iMessage عبر سطر أوامر imsg المحلي (macOS)
-   جسر وكيل لـ Pi في وضع RPC مع دفق الأدوات
-   الدفق والتقسيم للردود الطويلة
-   التوجيه متعدد الوكلاء لجلسات معزولة لكل مساحة عمل أو مرسل
-   مصادقة الاشتراك لـ Anthropic و OpenAI عبر OAuth
-   الجلسات: الدردشات المباشرة تنهار في جلسة `main` المشتركة؛ المجموعات معزولة
-   دعم دردشة المجموعات مع التنشيط بناءً على الإشارة
-   دعم الوسائط للصور والصوت والمستندات
-   خطاف اختياري لنسخ الملاحظات الصوتية
-   WebChat وتطبيق شريط قائمة macOS
-   عقدة iOS مع الاقتران، Canvas، الكاميرا، تسجيل الشاشة، الموقع، وميزات الصوت
-   عقدة Android مع الاقتران، علامة التبويب Connect، جلسات الدردشة، علامة التبويب الصوتية، Canvas/الكاميرا/الشاشة، بالإضافة إلى أوامر الجهاز، الإشعارات، جهات الاتصال/التقويم، الحركة، الصور، الرسائل القصيرة، وتحديث التطبيق

> **ℹ️** تمت إزالة مسارات Claude القديم، Codex، Gemini، و Opencode. Pi هو مسار وكيل البرمجة الوحيد.
