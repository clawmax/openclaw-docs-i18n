# skills

检查技能（捆绑 + 工作区 + 托管覆盖）并查看哪些符合资格或缺少要求。相关：

-   技能系统：[技能](../tools/skills.md)
-   技能配置：[技能配置](../tools/skills-config.md)
-   ClawHub 安装：[ClawHub](../tools/clawhub.md)

## 命令

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
