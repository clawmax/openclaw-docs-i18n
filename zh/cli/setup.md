# setup

初始化 `~/.openclaw/openclaw.json` 和智能体工作区。相关链接：

-   入门指南：[快速开始](../start/getting-started.md)
-   向导：[入门向导](../start/onboarding.md)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

通过 setup 命令运行向导：

```bash
openclaw setup --wizard
```
