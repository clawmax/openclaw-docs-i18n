# skills

Inspectez les compétences (groupées + espace de travail + remplacements gérés) et voyez ce qui est éligible par rapport aux prérequis manquants. Liens connexes :

-   Système de compétences : [Compétences](../tools/skills.md)
-   Configuration des compétences : [Configuration des compétences](../tools/skills-config.md)
-   Installations ClawHub : [ClawHub](../tools/clawhub.md)

## Commandes

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
