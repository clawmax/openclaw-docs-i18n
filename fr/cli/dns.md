# dns

Assistants DNS pour la découverte étendue (Tailscale + CoreDNS). Actuellement axé sur macOS + CoreDNS Homebrew. Liés :

-   Découverte de passerelle : [Découverte](../gateway/discovery.md)
-   Configuration de la découverte étendue : [Configuration](../gateway/configuration.md)

## Configuration

```bash
openclaw dns setup
openclaw dns setup --apply
```
